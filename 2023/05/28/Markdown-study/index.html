1.分级标题

语法：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
Eg:最后一个#后面必须有一个空格。

2.字体

语法：

*斜体文本*    
_斜体文本_
**粗体文本**
***粗斜体文本***

效果：

斜体文本
粗体文本

3.分割线

语法：

***

- - -

效果：

4.删除线、下划线

语法：

~~删除线~~
<u>下划线</u>

效果：

删除线

下划线

5.分条序列

5.1无序

语法:

* 第一行 
* 第二行

效果：

  • 第一行
  • 第二行

Eg:除了*，-和+都可以，记得空一格

5.1嵌套

语法：

1.第一行
- 第一点
    - 第二点
        - 第三点

效果：

1.第一行

  • 第一点
    • 第二点
      • 第三点

6.区块引用（常用）

6.1区块嵌套

语法：

>1
>>2
>>>3

效果：

1

2

3

6.2区块列表

语法：

> 区块中使用列表
> 1. 第一项
> 2. 第二项
>> + 第一项
>> + 第二项
>> + 第三项

效果：

区块中使用列表

  1. 第一项
  2. 第二项
    • 第一项
    • 第二项
    • 第三项

第二部分

Markdown进阶

1.markdown插入代码

插入一行代码

语法：

`int a = 0;`//英文版的数字1左边的那个符号。。。。

效果：

int a = 0;

插入代码块

语法：

​```紧跟着可以写上用的什么语言，也可以不用写。
int a = 1;
int b = 2;
​```结尾与开头一样

效果：

int a = 1;

int b = 2;

2.markdown插入链接

语法：

[链接名称](链接地址)
或者
<链接地址>

案例：[百度](https://www.baidu.com)

EG：符号全英文

效果：
百度

3.markdown插入图片

语法：

![图片描述，可写可不写，但是中括号要有](图片地址，本地链接或者URL地址。)


比如我此文章的图片：

![markdown](https://ss2.bdstatic.com/70cFvnSh_Q1YnxGkpoWK1HF6hhy/it/u=1613194509,3683268681&fm=26&gp=0.jpg)

4.markdown插入表格

每排单元格用|分开 表头与内容用-分开
语法：

|  表头   | 表头  | 表头 | 表头 |
| :---  | ---:  | :--: | ---- |
| 单元格  | 单元格 |单元格|单元格|
| 单元格  | 单元格 |单元格|单元格|

效果：

表头 表头 表头 表头
单元格 单元格 单元格 单元格
单元格 单元格 单元格 单元格

Eg:加：可以实现左对齐、右对齐、居中，不加默认为左对齐。

5.markdown支持HTML元素

语法：

使用 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Del</kbd> 重启电脑

效果：
使用 Ctrl+Alt+Del 重启电脑
