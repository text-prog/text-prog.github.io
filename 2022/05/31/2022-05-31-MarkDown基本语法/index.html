MarkDown基本语法

发表于2022-05-31|更新于2022-06-03
|字数总计:704|阅读时长:2分钟

基本语法

1.标题
注：# 后面必须保持空格（和使用 h1/h2 标签功能类似）||最多可以6级标题   

Eg："### h"  是三级标题     "######  h"是六级标题
2.TOC
注：根据标题生成目录
3.引用
格式：>引用的内容
同标题一样，共六级，可以嵌套
示例：>hello!

示例结果：

hello

4.语义标签(html)
示例：
  <i>斜体</i>  
  <b>加粗</b>  
  <em>强调</em>  
  <u>下划线</u>   
  <del>删除</del>  
  Z<sup>a</sup>  
  Z<sub>a</sub>  
  <kbd>Ctrl</kbd>
5.自定义语法（根据不同的语言配置不同的代码着色）

代码： 

var num = 0;
for (var i = 0; i < 5; i++) {num+=i;}
console.log(num);
6.插入链接
格式：[mingzi](链接 '描述')
示例：[百度](http://www.baidu.com/ '点击打开')

示例结果:百度

7.插入图片（本地&在线）

在线：

在线示例：![](https://img-blog.csdnimg.cn/20200522164615538.gif)

示例结果：

本地 

格式： ![描述](图片链接 "optional title")
示例1：![博客本人](E:/Blog/source/_posts/blog-images/MarkDown基本语法/author.jpg)
示例2：![](./blog-images/MarkDown基本语法/author.jpg)

示例1、2结果：实验验证均可以

思考，还有其他方法吗？
在上述markdown引用时，我们无法改变其他参数，例如大小等，所以解析来我们以html引用设置图像百分比

html：

格式：<img src="在线图片" width="" height="" />
示例：<img src="https://img-blog.csdnimg.cn/20200522164615538.gif" width=120 height=120 />
      <img src="./blog-images/MarkDown基本语法/author.jpg" width="120" height="120" />

示例结果：

8.插入视频
eg：注：Markdown 语法是不支持直接插入视频的
普遍的做法是 插入 HTML 的 iframe 框架，通过网站自带的分享功能获取，如果没有可以尝试第二种方法
第二是伪造播放界面，实质是插入视频图片，然后通过点击跳转到相关页面

示例：

<iframe height=300 width=300  src='http://player.youku.com/embed/XMjgzNzM0NTYxNg==' frameborder=0 ></iframe>

10.表格
示例：
|123|234|345|
|:-|:-:|-:|
|abc|bcd|cde|
|abc|bcd|cde|
|abc|bcd|cde|

结果

123 234 345
abc bcd cde
abc bcd cde
abc bcd cde
11.隐藏部分内容
方法1：在需要截断的地方加入：<!-- more -->
方法2：---
        title: 让首页显示部分内容
        date: 2020-10-13 15:34:10
        description: 这是显示在首页的概述，正文内容均会被隐藏。
        ---
随后重新hexo clean
